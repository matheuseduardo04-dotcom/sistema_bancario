## Sobre o Projeto

Este projeto é um desafio da [**Digital Innovation One (DIO)**](https://www.dio.me/), desenvolvido como parte da trilha de aprendizado em Python. O objetivo é simular as operações básicas de um sistema bancário utilizando apenas conceitos fundamentais da linguagem.

## Funcionalidades

- **[d] Depositar** — realiza um depósito de valor positivo na conta.
- **[s] Sacar** — efetua um saque respeitando o limite de R$ 500,00 por operação e o máximo de 3 saques diários.
- **[e] Extrato** — exibe o histórico de movimentações e o saldo atual.
- **[quantidade]** — informa a quantidade de saques ou depósitos realizados.
- **[q] Sair** — encerra o programa.

## Regras de Negócio

- Saldo inicial: `R$ 0,00`
- Limite por saque: `R$ 500,00`
- Número máximo de saques diários: `3`
- Valores inválidos (negativos ou zero) são rejeitados.

## Como Executar

Certifique-se de ter o Python 3 instalado em sua máquina e execute:

```bash
python sistemabancario.py
```

## Tecnologias

- Python 3
